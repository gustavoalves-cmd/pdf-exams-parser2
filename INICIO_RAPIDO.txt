📋 INSTRUÇÕES RÁPIDAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WINDOWS:
--------
1. Clique duas vezes em: iniciar_windows.bat
2. Aguarde abrir o navegador
3. Pronto! Use a interface

LINUX/MAC:
----------
1. Abra o terminal nesta pasta
2. Execute: ./iniciar_linux_mac.sh
3. Ou execute: python3 server.py
4. Acesse: http://localhost:5000

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📖 COMO USAR A INTERFACE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  UPLOAD DO PDF
   • Clique na área de upload
   • Ou arraste o PDF para a área
   • Apenas PDFs com texto selecionável

2️⃣  COLE O GABARITO
   • Formato 1: 32-A, 33-B, 34-C
   • Formato 2: 32: A (um por linha)

3️⃣  PROCESSAR
   • Clique em "Processar Prova"
   • Aguarde alguns segundos

4️⃣  BAIXAR EXCEL
   • Clique em "Baixar Arquivo Excel"
   • Revise e adicione imagens se necessário

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  IMPORTANTE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 SOLUÇÃO DE PROBLEMAS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Python não encontrado
   → Instale Python 3.8+ de python.org

❌ Erro ao instalar dependências
   → Execute: pip install -r requirements.txt

❌ Porta 5000 em uso
   → Edite server.py e mude para porta 8080

❌ Script não encontrado
   → Verifique a estrutura de pastas acima
